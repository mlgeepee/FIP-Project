Project Overview

What is it?

A modern web application designed to help students find and book hostel rooms efficiently. The platform features intelligent room recommendations, comprehensive filtering, and a seamless user experience.

Key Objectives

  • Simplify room search for students
  • Provide smart recommendations
  • Streamline booking process
  • Ensure secure user authentication
  • Deliver responsive design

Core Features

🔐 Authentication

  • User Registration
  • Secure Login
  • Session Persistence
  • Matric Number Validation

  • Price Range Filter

🤖 Smart Finder

  • AI Recommendations

  • Top 3 Results
